# TFC-Nuclear-Tech-Addon
A compability addon that bridges the gap between **HBM's Nuclear Tech Mod Community Edition** and **TerraFirmaCraft** for Minecraft 1.12.2.

**It's ready for survival**, but has some issues.

---

## Dependencies

This addon requires the following mods:

| Mod | Version |
|-----|---------|
| HBM's Nuclear Tech Mod CE | 2.5.0.5 |
| TerraFirmaCraft | 1.7.23.181 |
| MixinBooter | 10.7 |

---

## Installation

1. Ensure all dependencies are installed.
2. Download the `.jar` file from the [Releases](https://github.com/TheVoidedOnes/TFC-Nuclear-Tech-Addon/releases/tag/Release) page.
3. Place the downloaded `.jar` into your Minecraft `mods` folder.
4. Launch the game.

---

## Features & Issues

Many recipes have been modified to be compatible with TFC rocks/blocks/items/ores etc.
Metallurgy and ore processing recipes were transferred according to the mechanics from TFC.
Fully custom generation of all ores, but requires balancing.
There may be some minor issues with displaying recipes in JEI.

If you encounter any bugs or unexpected behavior, please **report them** by creating an [Issue](https://github.com/TheVoidedOnes/TFC-Nuclear-Tech-Addon/issues).
